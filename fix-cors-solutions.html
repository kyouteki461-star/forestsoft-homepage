Supabase CORS Error Solutions

重要: CORS (Cross-Origin Resource Sharing) は、ブラウザセキュリティ機能です。ローカルファイルから直接APIにアクセスすることはできません。以下のいずれかの方法で解決してください。

方法1: ローカルサーバーを起動する (推奨)

最も安全で推奨される方法です。Node.jsのhttp-serverを使います。

# npmをインストールしていない場合 npm install -g http-server # このHTMLファイルがあるディレクトリで cd /path/to/your/files http-server -p 8080 # ブラウザで http://localhost:8080 にアクセス

利点:

  • CORSエラーなし
  • HTTPS対応可能
  • 本番環境と同じ動作

方法2: VS Code Live Server拡張機能を使用

VS Codeの拡張機能を使ってローカルサーバーを起動します。

# 1. VS Codeで Ctrl+Shift+X で拡張機能を開く # 2. "Live Server" を検索してインストール # 3. HTMLファイルを右クリック → "Open with Live Server" # 4. ブラウザが自動的に開く

方法3: Pythonでローカルサーバー

Pythonがインストールされている場合。

# Python 3 python -m http.server 8080 # Python 2 python -m SimpleHTTPServer 8080 # ブラウザで http://localhost:8080 にアクセス

方法4: PHPでローカルサーバー

PHPがインストールされている場合。

# PHP内蔵のWebサーバー php -S localhost:8080 # ブラウザで http://localhost:8080 にアクセス

方法5: Supabase CORS設定を変更 (開発用)

SupabaseプロジェクトのCORS設定を変更できますが、セキュリティリスクがあります。

  1. Supabase Dashboardにログイン
  2. プロジェクト設定 → API → Configuration
  3. CORS originsに以下を追加:
    http://localhost:8080 https://*.github.io file:///*
警告: file:///* を許可すると、すべてのローカルファイルからアクセスできるようになり、セキュリティ上のリスクがあります。開発環境のみで使用してください。

方法6: プロキシサーバーを使用

ローカルプロキシサーバーを作成してCORSを回避します。

# proxy-server.js const express = require('express'); const { createProxyMiddleware } = require('http-proxy-middleware'); const app = express(); app.use('/supabase', createProxyMiddleware({ target: 'https://huatolfthmihcpzjpyipd.supabase.co', changeOrigin: true, pathRewrite: { '^/supabase': '' } })); app.listen(3000, () => { console.log('Proxy server running on port 3000'); });

JavaScriptコード内でベースURLを変更:

const SUPABASE_URL = 'http://localhost:3000/supabase';

おすすめのワークフロー

  1. 開発時: VS Code Live Server または http-server を使用
  2. 本番環境: GitHub Pagesなどのホスティングサービスを使用し、CORS設定を適切に構成
  3. 長期的なプロジェクト: ローカル開発環境（Vite, Create React Appなど）を使用

環境ごとの推奨設定

ローカル開発環境の場合:

# .env SUPABASE_URL=http://localhost:3000/supabase SUPABASE_ANON_KEY=your_anon_key

GitHub Pagesの場合:

# Supabase DashboardでCORSに追加 https://*.github.io https://kyouteki461-star.github.io
